# Advent of Code 2025

Since this year's advent of code is shorter than previous years I want to do it all from start to finish, always on the day that the puzzles come out. My additional personal goal is also to get every puzzle solution under 0.1 seconds. I will be using the command "perf stat -r100" to find the average performance of my solutions.
